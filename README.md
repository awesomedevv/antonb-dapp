# Antonb Dapp

Antonb web app with mixer features.

# Development

1. Clone this repo

2. Install dependencies by `yarn`

   ```base
   yarn install
   ```

3. Launch the UI
   ```base
   yarn run start:dapp
   ```

## Running local Antonb-tools

```bash
# run this to update the packages
yarn install --check-files
```
